When something is off, the operator has two levers. Both land on Solana. Both take effect on the next `guard()` call from your agent.

</Frame>

## Pause

**Reversible.** The on-chain status flips to `paused`. The session keypair stays intact in your env, but `agent_check` denies every action with `reasonCode: "agent_paused"`. Click **Unpause** and the agent resumes from exactly where it stopped.

Reach for pause when:
- You see a transaction you don't recognise and want a minute to look.
- A protocol you trade against is having an incident.
- You're deploying a model change and want the agent quiet while it lands.

## Revoke

**Terminal.** The on-chain status flips to `revoked` and the Swig session-key scope is destroyed. The session keypair file in your env is now a dead key: it has no authority on-chain, no matter who holds the secret. The API key still authenticates so straggler `report()` calls still write audit rows, but `agent_check` denies every action with `reasonCode: "agent_revoked"`.

You cannot un-revoke. If you need the agent back, register a new one.

Reach for revoke when:
- You believe the session key has been exfiltrated.
- The model behind the agent is being retired.
- You're shutting the agent down for good.

## What a denial looks like in code

```ts
try {
  await altheia.guard(
    { type: "transfer", asset: "USDC", amount: 1200, target: addr },
    runFn,
  );
} catch (err) {
  if (err instanceof PolicyDeniedError) {
    console.log(err.reasonCode); // "over_per_tx_cap" | "agent_paused" | "agent_revoked" | ...
    console.log(err.message);    // "Transfer 1,200 USDC exceeds 50 per-tx cap"
  }
}
```

Every denial writes an audit row. Revocations also emit a Solana memo on the agent's PDA, so the event is verifiable on Solscan without trusting Altheia.

## Common reason codes

| Code | Meaning |
| --- | --- |
| `over_per_tx_cap` | Action exceeds `asset_caps[mint].max_per_tx`. |
| `over_per_day_cap` | Action would push the rolling 24h total over `max_per_day`. |
| `program_not_in_scope` | The target program isn't in `allowed_programs`. |
| `destination_blocked` | The target address is in `blocked_destinations`. |
| `rate_limited` | Hit `max_actions_per_minute`. |
| `agent_paused` | Operator paused this agent. |
| `agent_revoked` | Operator revoked this agent. Permanent. |
| `policy_expired` | `expires_at` has passed. |
